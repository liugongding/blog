# 项目描述
本项目是一个极简的博客系统。展示页面提供了用户的读写操作即查看文章和评论文章。管理员可以对后台数据管理。
# 实现功能
分页展示、文件上传、登陆拦截、评论留言、部署云服务器。
# 技术栈
前端：html+css+jquery  
后端：spring+springMVC+mybatis  
IDE：eclipse
